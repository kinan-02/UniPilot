# Entrepreneurial Leadership Program (תכנית יזמות ומנהיגות)

**Administered by:** [[department-humanities-arts]] (Language Center / Humanities)
**Type:** Cross-faculty enrichment minor
**Credits:** 10 credits beyond home track requirements
**Available to:** All Technion BSc students

See also: [[technion-specializations]] for full program details.

## Description

A 10-credit cross-faculty minor in entrepreneurship and leadership, available to students in all Technion tracks. Courses include Hi-Tech in Israel, Entrepreneurship in Organizations, Legal Aspects of Business, and others.
