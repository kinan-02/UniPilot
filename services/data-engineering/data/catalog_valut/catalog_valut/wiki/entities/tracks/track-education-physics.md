## Special Rules

1. Teacher certification: Must pass התנסות קלינית בהוראה א' + ב' with average ≥ 75.
2. קורס סוגיות מתקדמות בהוראת הפיזיקה (02160005) נפתח אחת לשנתיים, לסירוגין עם קורס התפתחויות בהוראת הפיזיקה (02160004) — יש להירשם לקורס שייפתח ולקחת את השני בשנה העוקבת.
3. אפשר ומומלץ להחליף את הקורס כימיה פיסיקאלית (01240415) בקורס פיזיקה סטטיסטית ותרמית (01140036). הפרש הנקודות יחשב כבחירה מומלצת מרשימה ב'.
4. The program satisfies Ministry of Education requirements for grades 7–12 teaching license.

## נתונים בעברית

תכנית הלימודים בהוראת הפיזיקה מכשירה אנשי חינוך, מחנכים ומורים שיכולים ללמד פיזיקה באופן מעמיק וחוויתי בביה"ס העל יסודי ובמסגרות אחרות בחינוך הפורמלי והלא פורמלי. תוכנית הלימודים במסלול מקנה בסיס מדעי איתן דרך לימודי פיזיקה בפקולטה לפיזיקה ולימודי מתמטיקה ומקצועות מדעיים והנדסיים משיקים בפקולטות מדעיות והנדסיות אחרות בטכניון; ובסיס חינוכי איתן דרך לימודי חינוך ופדגוגיה בפקולטה לחינוך למדע וטכנולוגיה בטכניון ומפגשים מודרכים עם אנשי חינוך מובילים בפיזיקה ועם תלמידים במגמת הפיזיקה בבתי ספר. בוגרי המסלול משתלבים במערכת החינוך הפורמאלית והלא פורמאלית כמורים, מפתחי תוכן, ומדריכי מורים, ויכולים להמשיך ללימודים לתארים מתקדמים.

על מנת להשלים את התואר יש לצבור 155.0 נקודות לפי הפירוט הבא:
- 105.5 נק' מקצועות חובה
- 37.5 נק' מקצועות בחירה מומלצת
- 6.0 נק' מקצועות העשרה
- 4.0 נק' מקצועות בחירה כלל טכניונית
- 2.0 נק' חינוך גופני
